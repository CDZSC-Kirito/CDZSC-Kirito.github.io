A - I Scream

题意：

在日本，冰制品有一下4大类：
1，乳固形分含量15%以上，乳脂肪分含量8%以上的称为冰激凌
2，不满足第1种且乳固形分含量10%以上，乳脂肪分含量3%以上的称为冰牛奶
3，不满足前2种且乳脂肪分含量3%以上的称为乳酸冰淇淋
4，不满足前3种的称为冰点心
此外，乳固形分由無脂乳固形分和乳脂肪分2种组成
现在由1个产品的無脂乳固形分含量为A%，乳脂肪分含量为B%，问这个产品属于哪类

分析：

简单模拟，注意乳固形分含量=A%+B% 乳脂肪分含量=B%

C - Squared Error

题意：

给定一个长度为$N$数列$A$。
求数列$A$种两两元素差的平方之和。

分析：

考虑平方差公式分解：
$(X-Y)^{2} = X^{2} - 2XY + Y^{2}$

平方部分($X^{2}$)：数列中每个元素的平方都出现$N-1$次

乘法部分($-2XY$)：数列中每个元素都与这个元素之前的每个元素做一遍乘法

即元素$A_{i}$会与元素$A_{i-1}、A_{i-2}、…、A_{1}$做一遍 $\sum_{j=1}^{i-1} -2A_{i}A_{j}$

最终结果为$ans = \sum_{i=1}^{n}A_{i}^{2}-2A_{i}PreSum_{i-1}$

E - Mex Min

题意：

一个序列的$mex$定义为最小未出现的自然数

现在有一个长度为$N$数列$A=(A_{1}, A_{2}, A_{3},…,A_{N})$

对于所有$0\leq i \leq N-M$，都存在一个$mex_{i} = mex(A_{i+1}, A_{i+2}, A_{i+3}, …, A_{i+M})$。

总共有$N-M+1$个$mex_{i}$，求其中的最小值是多少。

分析：

由于题目求的是长度为$M$的连续子序列$mex$，考虑当前已知$mex_{i}$，如果求出$mex_{i+1}$。

$mex_{i+1}$对于$mex_{i}$去除了最左边的元素，加入了最右边的元素。

对于长度为$M$的$mex_{i}$，答案必定在$0 \leq mex_{i} \leq M$。

可以用线段树维护最小未出现过的元素，建立一颗区间大小为$M+1$的线段树。

树的叶子节点维护该数出现个次数，非子叶节点维护该区间的出现次数的最小值

$Node_{i} = min(Node_{Left}, Node_{Right})$

查找$mex$时如果左子树的最小值为0则走左子树，否正走右子树。

$mex_{i}$到$mex_{i+1}$的转移为：

去除了最左边的元素相当于最左元素出现次数-1

加入了最右边的元素相当于最右元素出现次数+1

D - Circle Lattice Points

题意：

在二维平面上有1个圆心为$(X,Y)$，半径为$R$的圆

问这个圆内（包括边界）有多少个整数坐标点。

分析：

固定一维，计算另外一维的2个交点，然后计算交点内的整数点数量即为当前的贡献，累计所有贡献即为答案。
注意浮点数精度问题，使用long double 和对输入的$R$增加$1e^{14}$来防止精度丢失导致答案计算错误。

E - Come Back Quickly

题意：

AtCoder国家由$N$座城市，编号为$1,…,N$，有$M$条有向道路，道路$i$表示从城市$A_{i}$前往城市$B_{i}$需要花费$C_{i}$分钟。

另外道路可能有自环(即存在$A_{i}=B_{i}$)，可能存在多条道路。

高橋君想在这个国家散步，定义一条正确散步路径为：从某个城市出发，经过至少1条道路，最后再回到出发点城市的路径。
现在要判断从各个城市出发，是否存在这样的正确散步路径，如果存在最少需要花费多少时间？

分析：

假设我们从城市$B_{i}$出发，有1条道路为$A_{i}$前往$B_{i}$，耗时$C_{i}$分钟。

则可以先求出从$B_{i}$前往$A_{i}$后，最后经由这条道路回到出发城市$B_{i}$，总的耗时为$Cost = Cost_{B->A} + C_{i}$。

如果有多条路径能到达出发城市$B_{i}$，则从这些耗时中取最小值即可。

通过dijstra算法求出当前出发城市$i$到各个城市的最短路，然后枚举所有能前往城市$i$的道路，通过最短路和道路耗时求出最终答案。

注意题目中有自环，自环也属于合法散步路线的情况。
